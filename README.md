Brave extension for summarizing and re-formatting:
(i) earnings calls / presentations
(ii) expert calls in AlphaSense, AlphaSights (but obviously extensible).

Creates HTML output, with ability to bookmark. 

Summary page includes bookmarks and history of summarized calls, with links to original transcript.

Direct extension settings include verbosity on 0-200 scale (0 is extreme bullet points, 200 is close to verbatim)
